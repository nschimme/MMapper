### Minimum System Requirements

- **Operating System:**
  - **Windows:** Windows 11
  - **macOS:** macOS 13 (Ventura) or newer
  - **Linux:** Ubuntu 22.04 LTS or newer
- **Processor:** Intel Sandy Bridge (2nd Gen Core i3/i5/i7) or newer / AMD equivalent.
- **Graphics:** OpenGL 3.3 or OpenGL ES 3.0 compatible GPU (Intel HD Graphics 3000, NVIDIA GeForce 400, AMD Radeon HD 5000 or newer).
- **Memory:** 1 GB RAM.
- **Disk Space:** 1 GB.
